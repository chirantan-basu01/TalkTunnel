# ChatApp

## *Chirantan Basu*

**ChatApp** is an app that is used to chat with friends.

## Features

The following features are implemented:
* [x] Sign In
* [x] Create New Account
* [x] Dark theme/ Light theme
* [x] Secure chat
* [x] Log out



## Tech Used
**Server**: Firebase Auth, Firestore 
**Client**: Flutter
